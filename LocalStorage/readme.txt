

Una aplicación web simple, elegante y funcional para gestionar tus tareas diarias.

 Características

-  Agrega nuevas tareas fácilmente.
-  Marca tareas como completadas con un clic.
-  Tus tareas se guardan automáticamente en el navegador (Local Storage).
- Interfaz limpia, moderna y responsive.

 Tecnologías

- HTML5
- CSS3 (Flexbox, responsive, variables)
- JavaScript Vanilla
- Local Storage

Archivos del proyecto

- `index.html`: Estructura principal de la app.
- `style.css`: Estilos modernos y responsivos.
- `script.js`: Lógica para manejar tareas y almacenamiento.
- `README.md`: Documentación del proyecto.

 Cómo usar

1. Clona o descarga este repositorio.
2. Abre el archivo `index.html` en cualquier navegador moderno.
3. ¡Empieza a organizar tu día!
