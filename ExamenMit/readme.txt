
Examen Práctico Único – HTML5, CSS y JavaScript (Nivel Medio)
Proyecto: “Gestor de Tareas Personales”

Este examen evalúa tus conocimientos en HTML5, CSS y JavaScript mediante la creación de una aplicación web interactiva
que permita al usuario gestionar una lista de tareas.
Deberás demostrar dominio en estructura semántica, manejo del DOM, eventos, validaciones y uso de Local Storage.

Requisitos técnicos
1. Estructura semántica (HTML5)
Utiliza correctamente las etiquetas header, nav, main, section, article, y footer.
Incluye una barra de navegación con al menos dos enlaces (por ejemplo: Inicio y Acerca de).
Define los metadatos esenciales:
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<meta name="description" content="Aplicación web para gestionar tareas personales">

2. Formulario HTML5
Crea un formulario para agregar nuevas tareas que contenga los siguientes campos:
Título de la tarea (required)
Fecha límite (type="date", required)
Prioridad (select con opciones: baja, media, alta)
// Aplica validaciones nativas de HTML5 (required, type, min, max) donde corresponda.

3. Estilos (CSS)
Aplica un esquema de colores coherente y legible.

4. Lógica con JavaScript
usa getElementById, querySelector para acceder y modificar elementos del DOM.
Implementa eventos con addEventListener para:
Añadir una nueva tarea.
Marcar una tarea como completada.
Muestra un mensaje dinámico (por ejemplo, “¡Tarea añadida con éxito!”) cuando se agrega una tarea.

5. Almacenamiento (Local Storage)
Guarda la lista de tareas en Local Storage.
Al recargar la página, las tareas guardadas deben mostrarse automáticamente.

6. Accesibilidad y usabilidad
Asegura que todas las etiquetas estén correctamente asociadas a sus campos (label for).
Incluye descripciones claras y elementos accesibles (por ejemplo, texto alternativo en íconos o imágenes).
Mantén el código ordenado y con indentación consistente.
Entrega final

El proyecto debe consistir en tres archivos:
index.html
style.css
script.js

Debe poder abrirse en cualquier navegador moderno sin requerir servidor adicional.
